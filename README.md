# EyeCalibrationDemo

Qt C++ 暗瞳法眼动校准示例，包含头部运动补偿。

## 主要算法说明
- **特征构成**：利用瞳孔中心与角膜反射点的相对向量 `(dx, dy)`，并附加头部三维位移 `(hx, hy, hz)` 以及偏置项，共 6 个特征。
- **拟合方式**：通过最小二乘求解 `(X^T X) w = X^T y`，分别得到屏幕 `x`、`y` 两个映射向量。实现中使用简易高斯消元并配合主元选取以避免奇异。
- **补偿思路**：在预测时同时输入头部坐标，模型会学习头动对瞳孔向量的影响，从而在新姿态下仍能正确映射。

## 构建与运行
```bash
cmake -S . -B build
cmake --build build
./build/EyeCalibration
```

> 若环境未安装 Qt6，请先安装 `qt6-base-dev` 或使用 Qt 提供的预编译套件。

## Demo 演示
- **模拟采集并标定**：生成 3×3 屏幕网格样本，叠加随机头部位移，完成校准。
- **测试预测**：使用新的头部位置与瞳孔观测，输出补偿后的预测屏幕坐标。
